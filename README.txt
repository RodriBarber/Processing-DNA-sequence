Aplication v1.1 (2025)
Program written in python / Mac OS X / Linux / Windows

1.Introduction

A DNA sequence processing application is implemented, which allows for various tasks, such as pattern searching, 
nucleotide content analysis, gene identification and translation into proteins, or mutation simulation. A FASTA 
file is required as an input parameter (with one or more sequences).

2. Execution

To start the application, all files must be in the same folder. Then go to the file called 'main_body' and run the script.
The results will appear in the terminal and can finally be saved to a text file.
